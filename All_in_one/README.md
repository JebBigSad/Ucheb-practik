# Ucheb-practik — Многофункциональный веб-сервис

**Учебная практика УП.02 | Веб-приложение "всё в одном"**

---

## О проекте

Ucheb-practik — это веб-приложение, объединяющее пять полезных инструментов в одном интерфейсе. Проект разработан в рамках учебной практики УП.02.

### Включенные модули:

| Модуль | Описание |
|--------|----------|
| 🧮 Калькулятор | Базовые арифметические действия (+, -, *, /) |
| 🎮 Крестики-нолики | Классическая игра для двух игроков |
| 📅 Календарь | Просмотр текущего месяца, навигация по месяцам |
| 🕒 Часы | Цифровые часы с обновлением в реальном времени |
| 🌦 Погода | Прогноз погоды через OpenWeatherMap API |

---

## Технологии

- HTML5 — семантическая разметка
- CSS3 — Flexbox, Grid, медиа-запросы
- JavaScript ES6 — логика приложения
- OpenWeatherMap API — данные о погоде
